# Scores

This directory contains the musical scores of **Shinso Ondo**
(*Shinso Yoitoko Sanpomichi*).

## Contents

The score consists of seven verses with a four-measure instrumental interlude after Verse 4.

The published score is intended as the official reference version for performance and preservation.

# 楽譜

このディレクトリには、**「新荘音頭」**
（副題：「新荘よいとこ散歩道」）の楽譜を収録しています。

## 内容

楽譜は全7番で構成され、4番の後に4小節の間奏があります。

公開している楽譜は、演奏および保存のための公式な参照版です。

## 作曲・編曲
