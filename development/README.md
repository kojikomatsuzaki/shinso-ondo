# Development

## 日本語

このフォルダには、「新荘音頭」デジタルアーカイブの制作・管理に関する資料を収録しています。

ここに保存するファイルは、公開作品そのものではなく、アーカイブを構築・運用するための補助資料です。

例

- 制作メモ
- 作業手順書
- リポジトリ設計資料
- メタデータ設計資料
- GitHub運用資料
- 公開前チェックリスト
- 将来の改善計画
- その他、開発・保守に関する資料

公開作品（音源・歌詞・楽譜・パンフレット等）は、それぞれのフォルダで管理します。

## このフォルダの目的

本リポジトリでは、

- 文化資料
- 構造化メタデータ
- 開発・運用資料

を分離して管理しています。

これにより、

- 公開コンテンツ
- 管理情報
- 制作プロセス

をそれぞれ独立して更新・保存できる構成としています。

# English

This directory contains development and maintenance documents for the **Shinso Ondo Official Digital Archive**.

These materials describe how the archive is designed, managed, and maintained rather than the cultural resources themselves.

Typical contents include:

- development notes
- repository documentation
- metadata design notes
- workflow documents
- publication checklists
- maintenance plans
- future enhancement proposals

Public materials such as audio recordings, lyrics, scores, and publications are stored in their respective directories.

## Purpose

This repository separates:

- cultural resources
- structured metadata
- development documentation

This organization improves maintainability, transparency, and long-term preservation of the archive.
