# このリポジトリについて

このリポジトリは、小松崎浩司（Hiroshi Komatsuzaki）が管理する「新荘音頭」の公式デジタルアーカイブです。

GitHubのバージョン管理機能を活用し、次のことを目的として整備しています。

- 資料の保存
- 更新履歴の管理
- 修正前を含む版の保存
- マルチメディア資料の公開
- メタデータの整備
- 継続的な公開・改善
- 研究・教育・地域活動における再利用

公開後に資料やクレジットを修正した場合でも、変更前の状態や変更内容、変更日時を履歴として確認できます。

# English

## About

This repository is the official digital archive of **Shinso Ondo**, a local folk dance song performed during the **Shinso Summer Festival** in the Shinso District of Mito City, Ibaraki, Japan.

The archive is maintained by **Hiroshi Komatsuzaki**. It collects, organizes, preserves, and provides access to audio recordings, lyrics, musical scores, pamphlets, metadata, and production records related to Shinso Ondo.

---

## About This Repository

This repository uses GitHub’s version control features for the following purposes:

- Preservation of materials
- Management of revision history
- Preservation of previous versions
- Publication of multimedia materials
- Metadata management
- Continuous publication and improvement
- Reuse in research, education, and community activities

Even when materials or credits are revised after publication, previous states, the details of changes, and the dates of those changes remain traceable in the repository history.

## Features of This Archive

This digital archive combines:

- Version control with GitHub
- Documentation using Markdown
- Metadata management using YAML
- Publication of multimedia materials in formats such as PDF, PNG, and WAV
- Version management through GitHub Releases
- Promotion of reuse through open licensing

Through these methods, the archive aims to be:

**“A digital archive that is not merely preserved, but actively used and continuously developed.”**

The archive preserves not only completed materials, but also records of how they have been organized, revised, and developed over time.
