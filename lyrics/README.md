# Lyrics

This directory contains the lyrics of **Shinso Ondo**  
(*Shinso Yoitoko Sanpomichi*).

## Contents

The song consists of seven verses.

An instrumental interlude of four measures is inserted after Verse 4.

## Lyricists

- Verses 1, 2, and 5: Ichiro Kawamata
- Verses 3, 6, and 7: Noboru Koami
- Verse 4: Yohei Kunugisawa

## Special Readings

The following place names are read as indicated:

- 七曲坂：ななまがり
- 馬口労町：ばくろちょう

# 歌詞

このディレクトリには、**「新荘音頭」**
（副題：「新荘よいとこ散歩道」）の歌詞を収録しています。

## 構成

歌詞は全7番で構成されています。

4番の後に、4小節の間奏があります。

## 作詞

- 1番・2番・5番：川又 一郎
- 3番・6番・7番：小網 昇
- 4番：椹沢 庸平

## 特殊な読み

地名の読みは、次のとおりです。

- 七曲坂：ななまがり
- 馬口労町：ばくろちょう
